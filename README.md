### 项目说明
> 该项目基于gulp构建，请勿修改构建脚本

### 项目结构
```
├── README.md
├── dist            # 构建输出地址
├── gulpfile.js     # gulp构建脚本
├── src             # 开发文件目录
├── node_modules    # npm包
└── package.json    
```
### 开发

##### 安装依赖包
```
npm install 或 yarn
```

##### 本地开发
```
npm start （ 或 yarn start ）
预览地址：
localhost:8001
```

##### 构建
```
npm run build ( 或 yarn run build)
```
